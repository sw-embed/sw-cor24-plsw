Implemented DCL declaration parser (src/parser.h) with support for scalar types (INT(8/16/24), BYTE, CHAR, PTR, WORD, BIT), array dimensions, storage classes (STATIC/AUTOMATIC/EXTERNAL/BASED), INIT attributes (numeric and string literals), level-based record declarations, and DECLARE synonym. Added TOK_INIT keyword. Enhanced AST nd_print for DCL attributes. 19-test suite validates all forms on COR24 emulator. Used arena allocation for identifier names to avoid stack overflow in nested record parsing context.
