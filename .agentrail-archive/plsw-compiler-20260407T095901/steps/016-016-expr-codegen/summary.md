Implemented expression code generation (codegen.h) with literal loading (lc/la), variable load/store (fp-relative and static), binary arithmetic (add/sub/mul), software division (__plsw_div), all comparison operators (ceq/cls + carry), unary negate/NOT, assignment codegen, and register allocation with stack spill. 19 test cases verified on COR24 emulator. Simple-RHS optimization avoids push/pop when right operand is literal or variable.
