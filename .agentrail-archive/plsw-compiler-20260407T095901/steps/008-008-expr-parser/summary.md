Implemented precedence-climbing expression parser with 11 precedence levels (logical OR/AND, bitwise OR/XOR/AND, equality, comparison, shift, additive, multiplicative, unary). Added NOT/OR/AND keyword tokens. Parser handles literals, identifiers, parenthesized sub-expressions, function calls with args, ADDR(var), field access (.field), pointer deref (->field). 30 test cases validate all operators, precedence, associativity, nested calls, and complex expressions. All tests pass on COR24 emulator.
