Define AST node types and implement a pre-allocated node pool (256 nodes). Node kinds: NODE_PROGRAM, NODE_DCL, NODE_PROC, NODE_IF, NODE_DO_WHILE, NODE_DO_COUNT, NODE_ASSIGN, NODE_BINOP, NODE_UNOP, NODE_CALL, NODE_RETURN, NODE_LITERAL, NODE_IDENT, NODE_ASM_BLOCK, NODE_FIELD_ACCESS, NODE_ARRAY_ACCESS, NODE_ADDR, NODE_DEREF, NODE_BLOCK. Each node has: kind, type_info, storage_class, level, left/right/next pointers, name, int_val. Implement alloc_node() and tree-building helpers. Test: allocate nodes and verify pool management.
