Implemented procedure prologue/epilogue code generation. Added cg_stmt(), cg_block(), cg_proc(), and cg_program_procs() to codegen.h. Standard COR24 calling convention: push fp/r2/r1, mov fp,sp prologue; mov sp,fp, pop r1/r2/fp, jmp (r1) epilogue. Stack allocation for locals via sub sp. NAKED procedures skip prologue/epilogue. RETURN evaluates expr into r0 before epilogue. Added str_find() to str.h. Validated with 10-test suite on emulator: empty proc, return value, local vars, params at fp+9, NAKED, multiple procs, multi-locals, FREESTANDING, byte locals, globals+proc.
