Implemented PL/SW lexer (src/lexer.h) scanning source buffers character-by-character into tokens. Handles identifiers, numbers, single-quoted strings, all operators (including multi-char <=, >=, !=, <<, >>, ->), comments (/* */), whitespace, colons, and macro/directive prefixes. Added 10 test cases all passing. Discovered tc24r struct codegen bugs (filed as sw-embed/sw-cor24-tinyc#1): struct member resolution across multiple types panics, and struct pointer access produces garbled output. Used parallel arrays and global scalars as workaround -- TODO revert once tc24r is fixed.
