Implement the PL/SW lexer. Scan input buffer character by character producing tokens. Handle: identifiers (uppercase/lowercase), decimal numbers, string literals (single-quoted), operators (including multi-char: <=, >=, \!=, <<, >>), punctuation, semicolons. Skip whitespace. Skip comments (/* ... */ style). Store token type, value, and name text. Test: tokenize sample PL/SW fragments and dump token stream via UART.
